# Code of Conduct

## Our Pledge

We pledge to make participation in this project a welcoming and harassment-free experience
for all contributors, regardless of background, identity, or experience level.

## Standards

- Cite all astronomical data sources (MPC, JPL, ESA/NASA ephemerides)
- Do not fabricate or misrepresent observational data
- Acknowledge co-observers and collaborators
- Respect indigenous astronomical knowledge when referenced

## Interpretação RAFAELIA

O projeto enquadra cada fenômeno celeste como parte de uma **equação viva**:

```
VAZIO → VERBO → CHEIO → RETROALIMENTAÇÃO → NOVO VAZIOₙ
```

Cada cometa representa um ciclo: nascimento (periélio), travessia (coma), retorno (afélio).
